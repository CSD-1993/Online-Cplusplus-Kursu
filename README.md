+ Bu kurs [C ve Sistem Programcıları Derneği](http://www.csystem.org/) ve _Plepa Eğitim Hizmetleri_ işbirliği ile düzenlenmektedir.
+ Kurs _12 Eylül 2020 tarihinde_ başlayacaktır.
+ Dersler _Cumartesi ve Pazar_ günleri saat _14.00-18.00_ arasında yapılacaktır.
+ Kursun toplam süresi _162_ saattir.
+ Programın toplam ücreti 3.500 TL_'dir. Öğrenci olanlara (öğrencilik durumunu belgelemeleri kaydıyla) _% 10_ indirim uygulanmaktadır. Katılımcılar, eğitim programı ücretini dilerlerse 2 taksitte ödeyebilirler.
